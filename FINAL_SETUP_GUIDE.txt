🆘 IF SOMETHING GOES WRONG
═══════════════════════════════════════════════════════════════════════════

Error: "npm: command not found"
→ Node.js not installed properly
→ Download and reinstall from nodejs.org
→ Restart your computer after installation

Error: "Module not found"
→ npm install didn't complete
→ Make sure you waited for "added XXX packages" message
→ Try: npm install --legacy-peer-deps --force

Error: "Port 3000 already in use"
→ Another app using port 3000
→ Close other applications or:
→ Type in backend window: npm run dev -- --port 3001

Error: Browser won't load http://localhost:5173
→ Frontend not running properly
→ Check frontend window shows "Local: http://localhost:5173"
→ Try refreshing browser (Ctrl+R)
